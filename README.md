# Lamp Architect

Configurador paramétrico 3D de lámparas artesanales con hilo de papel Kraft. Permite ajustar geometría, trama y calibre del hilo en tiempo real, con visualización interactiva y cálculo estructural automático.

## Stack

- **React 18** (vía ESM / import maps)
- **Three.js** + React Three Fiber + Drei
- **Tailwind CSS** (CDN)
- **Babel** (compilación JSX en navegador)
- **Lucide React** (iconografía)

## Funcionalidades

- Visualización 3D interactiva con rotación automática y controles orbitales
- Ajuste paramétrico de altura, radios (base y superior) y curvatura
- Configuración de vueltas de trama y calibre del hilo
- Motor de cálculo estructural: longitud total de hilo, masa, tensión y factor de seguridad
- Indicador visual de seguridad estructural (verde/rojo)
- Interfaz con panel lateral tipo glass-morphism y estadísticas flotantes
- Exportación de reporte (simulado)
